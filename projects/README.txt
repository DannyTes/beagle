-------------
led-panel-v01
-------------

Code to drive a single SparkFun / Adafruit 32x32 LED panel using
a BeagleBone Black and a ValentFX LogiBone FPGA board. 

-------------
led-panel-6up
-------------

Code to drive up to six SparkFun / Adafruit 32x32 LED panels using
a BeagleBone Black and a ValentFX LogiBone FPGA board.
